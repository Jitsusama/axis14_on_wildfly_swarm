# Overview

This repository holds a simple example of an issue I've been having creating
uber-jars via Wildfly Swarm against a (very) simple Apache Axis 1.4 SOAP RPC
servlet application.

# Issue Reproduction

To reproduce the issue, run `mvn package` from this directory, and then try to
run the created jar file in the `target` directory.
